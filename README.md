# Eclipse Romania Mafii Free Server

Welcome to the Eclipse Romania Mafii Free Server project! This repository contains the source code for the website dedicated to the FiveM server "Eclipse Romania Mafii Free". 

## Project Overview

The Eclipse Romania Mafii Free server is a community-driven FiveM server that offers an immersive role-playing experience. Our server features various factions, including mafias, and provides a unique environment for players to engage in thrilling gameplay.

## Features

- **Community Focused**: Join a vibrant community of players who share a passion for role-playing.
- **Custom Scripts**: Enjoy unique gameplay mechanics with our custom scripts designed for an enhanced experience.
- **Active Administration**: Our dedicated team ensures a fair and enjoyable environment for all players.
- **Regular Updates**: We continuously improve the server with regular updates and new content.

## Getting Started

To set up the project locally, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/eclipse-romania-mafii-free-site.git
   ```
2. Navigate to the project directory:
   ```
   cd eclipse-romania-mafii-free-site
   ```
3. Open `index.html` in your web browser to view the website.
